Types of HTML Elements

Block-level Elements

Block-level elements take up the full width available and always start on a new line. Common block-level elements include:

  • <div>
  • <h1> to <h6>
  • <p>
  • <section>
            
                <div>This is a div</div>
                <h1>Heading 1</h1>
                <p>This is a paragraph.</p>

Inline Elements

Inline elements do not start on a new line and only take up as much width as necessary. Common inline elements include:

  • <span>
  • <a>
  • <img>
  • <strong> and <em>
            
                <span>This is a span</span>
                <a href="#">This is a link</a>
                <img src="image.jpg" alt="Image">
